+								wogame-loganalyser															+
+																											+
=============================================================================================================
runPackageInfoAnalyser.sh用于解析打包平台的回执文件，存入数据库。
runLogAnalyser.sh和runDownloadAnalyser.sh用于解析沃游戏H5页面的日志信息（PV，UV等）。目前该部分功能已接入联通Log服务器，由大数据负责解析日志。
runReportAnalyser.sh解析打包平台每天的打包情况，为管理后台提供报表数据。
